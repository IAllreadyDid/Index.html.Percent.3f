This Patch provides you with a working date command for the year 2000.

Important change: Setting date after Dec. 31 1999 is possible

Sun3:

Patch Installation Instructions: 

1) As root, save the original date file:
   mv /bin/date /bin/date.FCS
   
2) Copy the new date file from the patch directory:
   cp date /bin/date
   
3) chmod 755 /bin/date
   chown root.staff /bin/date
